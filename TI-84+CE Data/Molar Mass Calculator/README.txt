Requires Textlib, can be found on Ticalc.net or cemetech.net

Given a compound in the format BrSO4, returns the molar mass for each element {Br,S,O},
multiplied by the number following it {Br*1,S*1,O*4},
and the total molar mass (Br*1+S*1+O*4)

Does not accept parenthesis, so numbers must be distributed, (A2B3)2 = A4B6
